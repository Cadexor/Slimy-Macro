# Slimy-Macro
An ahk base macro for Slime RNG

# Instructions
1. Download the zip file.
2. Extract the compressed folder.
3. Run the "SlimyMacro" file.

# Features
1. Auto rebirth
2. Auto buy next zone
3. Auto use the slime gun

# Requirements
1. Have AutoHotkey v2 installed.
2. You must have the teleporter to use the auto buy zone feature.

# Note
Turn off any cutscene settings for best effect, things like slime cutscene and fruit cutscene can be turned off in the game's settings.
Also use the lowest graphic settings if you want to consume less resources.
